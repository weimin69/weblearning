## 使用建议

### 从v1升级到v2
1. 备份原有配置和数据
2. 安装新依赖: `pip install -r requirements.txt`
3. 复制配置文件: `cp config_example.py config.py`
4. 修改配置文件中的课程URL
5. 使用新启动脚本: `./start_all_platforms.sh`

### 兼容性说明
- v2版本完全兼容v1的课程URL
- v2新增的配置文件系统可选使用
- 原版脚本 `auto_course.py` 仍然可用
- 建议新用户直接使用v2版本
